
##  Mathematical / Vectorized Operations

###  Theory

One of NumPy's greatest powers is **vectorized operations** — applying a mathematical operation to **every element** of the array **without using a loop**.

###  Arithmetic Operations

You can use standard operators `+`, `-`, `*`, `/`, `**` (power), `//` (floor division), `%` (modulo) directly on arrays.

```python
arr = np.array([1, 2, 3, 4])

arr + 10   → [11, 12, 13, 14]   # Add 10 to every element
arr * 3    → [ 3,  6,  9, 12]   # Multiply every element by 3
arr ** 2   → [ 1,  4,  9, 16]   # Square every element
```

### ➕ Element-wise Operations Between Two Arrays

When two arrays have the **same shape**, operations happen **element by element**:

```python
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

a + b  → [5, 7, 9]    # 1+4, 2+5, 3+6
a * b  → [4, 10, 18]  # 1*4, 2*5, 3*6
```

###  Built-in Math Functions

NumPy has powerful built-in math functions:

| Function | What it does |
|----------|--------------|
| `np.sqrt(arr)` | Square root of each element |
| `np.abs(arr)` | Absolute value of each element |
| `np.exp(arr)` | e raised to the power of each element |
| `np.log(arr)` | Natural logarithm of each element |
| `np.round(arr, n)` | Round to n decimal places |
```python
# ============================================================
# TOPIC 9 EXAMPLE: Mathematical / Vectorized Operations
# ============================================================

import numpy as np

arr = np.array([1, 2, 3, 4, 5])
print("Original Array:", arr)
print()

# --- Operations with a single number (scalar) ---
print("arr + 10 :", arr + 10)    # Add 10 to each element
print("arr - 1  :", arr - 1)     # Subtract 1 from each
print("arr * 3  :", arr * 3)     # Multiply each by 3
print("arr / 2  :", arr / 2)     # Divide each by 2
print("arr ** 2 :", arr ** 2)    # Square each element
print("arr % 2  :", arr % 2)     # Modulo (remainder when divided by 2)

print()

# --- Element-wise operations between two arrays ---
a = np.array([10, 20, 30, 40])
b = np.array([1,  2,  3,  4])

print("a:", a)
print("b:", b)
print("a + b:", a + b)    # [11, 22, 33, 44]
print("a - b:", a - b)    # [9, 18, 27, 36]
print("a * b:", a * b)    # [10, 40, 90, 160]
print("a / b:", a / b)    # [10.0, 10.0, 10.0, 10.0]
```

```python
# ============================================================
# TOPIC 9 EXAMPLE: Built-in Math Functions
# ============================================================

import numpy as np

arr = np.array([4, 9, 16, 25, 36])
print("Array:", arr)
print()

# Square root
print("np.sqrt(arr)  :", np.sqrt(arr))

# Absolute value
neg_arr = np.array([-5, -3, 0, 2, 7])
print("np.abs(neg_arr):", np.abs(neg_arr))

# Rounding
decimals = np.array([1.234, 5.678, 9.012])
print("np.round(decimals, 1):", np.round(decimals, 1))

print()

# --- Practical Example: Grade Percentage Boost ---
raw_scores = np.array([70, 80, 55, 90, 65])
print("Raw Scores      :", raw_scores)

# Add 5 bonus marks
boosted = raw_scores + 5
print("After +5 bonus  :", boosted)

# Calculate percentage out of 110 (boosted max)
percentage = np.round((boosted / 110) * 100, 2)
print("Percentage (%)  :", percentage)
```

##   Aggregation Functions

###  Theory

**Aggregation** means summarizing a collection of numbers into a **single value**.  
NumPy provides powerful functions to compute statistics over an array.

| Function | What it computes |
|----------|-----------------|
| `np.sum()` | **Sum** of all elements |
| `np.mean()` | **Average (Mean)** of all elements |
| `np.min()` | **Minimum** (smallest) value |
| `np.max()` | **Maximum** (largest) value |
| `np.median()` | **Median** (middle value when sorted) |
| `np.std()` | **Standard Deviation** (spread of data) |
| `np.var()` | **Variance** (square of std deviation) |

---

###  Quick Definitions

- **Mean**: Add all values, divide by count. Example: Mean of [60, 80, 100] = (60+80+100)/3 = 80
- **Median**: Sort the values, pick the middle one. Example: Median of [60, 80, 100] = 80
- **Std Dev**: How far (on average) each value is from the mean. **Small std = data is close together, Large std = data is spread out**
- **Variance**: Standard Deviation squared. Another way to measure spread.

---

###  Aggregation Along an Axis (for 2D arrays)

For 2D arrays, you can compute aggregations along:
- `axis=0` → **Down each column** (row-wise collapse)
- `axis=1` → **Across each row** (column-wise collapse)

```python
arr = [[1, 2, 3],
       [4, 5, 6]]

np.sum(arr, axis=0)  → [5, 7, 9]    # Sum of each column
np.sum(arr, axis=1)  → [6, 15]      # Sum of each row
```
```python
# ============================================================
# TOPIC 10 EXAMPLE: Aggregation Functions (1D)
# ============================================================

import numpy as np

# Student test scores
scores = np.array([72, 85, 90, 60, 78, 95, 88, 55, 70, 82])
print("Scores:", scores)
print()

print("Sum      :", np.sum(scores))              # Total marks
print("Mean     :", np.mean(scores))             # Average marks
print("Minimum  :", np.min(scores))              # Lowest score
print("Maximum  :", np.max(scores))              # Highest score
print("Median   :", np.median(scores))           # Middle score (sorted)
print("Std Dev  :", np.round(np.std(scores), 2)) # How spread out the scores are
print("Variance :", np.round(np.var(scores), 2)) # Variance (std² )

print()
# Finding index of min and max
print("Index of Min score:", np.argmin(scores))  # Who got lowest?
print("Index of Max score:", np.argmax(scores))  # Who got highest?
```

```python
# ============================================================
# TOPIC 10 EXAMPLE: Aggregation on 2D Arrays (with axis)
# ============================================================

import numpy as np

# 3 students, 4 subjects
marks = np.array([
    [80, 90, 70, 85],   # Student 0: Maths, Science, English, Hindi
    [60, 75, 88, 92],   # Student 1
    [95, 85, 78, 70]    # Student 2
])

print("Marks Table (rows=students, cols=subjects):")
print(marks)
print()

# Total of everything
print("Overall Total  :", np.sum(marks))
print("Overall Mean   :", np.mean(marks))

print()

# axis=1 → sum across each ROW (total per student)
print("Total per Student (axis=1):", np.sum(marks, axis=1))

# axis=0 → sum down each COLUMN (total per subject)
print("Total per Subject (axis=0):", np.sum(marks, axis=0))

print()

# Average per student
print("Avg per Student (axis=1) :", np.round(np.mean(marks, axis=1), 2))

# Average per subject
print("Avg per Subject (axis=0) :", np.round(np.mean(marks, axis=0), 2))
```
