## 📖 Concept Explanation

### Why do we explore data?

Imagine you're a store manager. Before making any business decisions, you'd want to know:
- How many transactions happened?
- What products were sold?
- What was the highest and lowest sale?
- Are there any missing entries in the records?

**Data Exploration** helps us answer these basic questions. It's like a health checkup for your data.

### What tools do we use?

We use a Python library called **Pandas**. Pandas gives us special functions (methods) to quickly explore any dataset:

| Method | What It Does |
| :--- | :--- |
| `.head()` | Shows the first 5 rows |
| `.tail()` | Shows the last 5 rows |
| `.shape` | Tells us (rows, columns) count |
| `.columns` | Lists all column names |
| `.dtypes` | Shows the data type of each column |
| `.info()` | Gives a complete summary (types + missing values) |
| `.describe()` | Gives statistics for number columns (mean, min, max, etc.) |
| `.value_counts()` | Counts how many times each value appears |
| `.unique()` | Shows all unique (distinct) values |
| `.nunique()` | Counts how many unique values exist |
| `.isnull().sum()` | Counts missing values in each column |
| `.sort_values()` | Sorts the data by a column |

### What is our dataset about?

We are using a **Sales Data** file (`sales_data.xlsx`) that contains 100 transactions from a retail store. Each row represents one transaction — what was sold, when, at what price, and how much revenue it generated.

print("Product Transaction Counts:")

most_sold_count = df['Product'].value_counts().values[0]
print(f"Most sold product: {most_sold} ({most_sold_count} transactions)")

# Which product was sold the LEAST?
least_sold = df['Product'].value_counts().index[-1]
least_sold_count = df['Product'].value_counts().values[-1]
print(f"Least sold product: {least_sold} ({least_sold_count} transactions)")
