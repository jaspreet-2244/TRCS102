

##  Topic : NumPy with Pandas

###  Theory

**Pandas** is a library built **on top of NumPy**.  
Under the hood, Pandas DataFrames and Series store data as NumPy arrays.

This means NumPy and Pandas **work together seamlessly**:
- You can convert Pandas data → NumPy arrays
- You can apply NumPy functions on Pandas columns
- You can create Pandas DataFrames from NumPy arrays

###  Key Conversions

| Task | Code |
|------|------|
| Pandas Series → NumPy Array | `series.to_numpy()` or `series.values` |
| DataFrame column → NumPy | `df['col'].to_numpy()` |
| NumPy Array → Pandas Series | `pd.Series(numpy_array)` |
| NumPy Array → DataFrame | `pd.DataFrame(numpy_array, columns=['a','b'])` |

###  Why does this matter?

- You can use fast NumPy functions on Pandas data
- Understanding this connection helps you write cleaner, faster code
- Most Data Science workflows mix both NumPy and Pandas together
```python
# ============================================================
# TOPIC 9 EXAMPLE: NumPy ↔ Pandas Conversion
# ============================================================

import numpy as np
import pandas as pd

# --- Create a DataFrame using NumPy array ---
np.random.seed(10)
data = np.random.randint(50, 100, size=(5, 4))

df = pd.DataFrame(data, columns=['Maths', 'Science', 'English', 'Computer'])
df.index = ['Alice', 'Bob', 'Charlie', 'Diana', 'Eve']

print("=== Pandas DataFrame created from NumPy Array ===")
print(df)

print()

# --- Convert DataFrame column to NumPy array ---
# maths_np = df['Maths'].to_numpy()
# print("Maths column as NumPy array:", maths_np)
# print("Type:", type(maths_np))

print()

# --- Apply NumPy operations on a Pandas column ---
print("Max Maths score :", np.max(df['Maths']))
print("Mean Maths score:", np.mean(df['Maths']))
print("Std Dev         :", np.round(np.std(df['Maths']), 2))
```
